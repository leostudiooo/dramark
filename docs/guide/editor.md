# 编辑器与工具

## VSCode 扩展

🥳 DraMark 的 VS Code 扩展已上线 [Marketplace](https://marketplace.visualstudio.com/items?itemName=LeoLi.dramark)！

功能包括：
- 语法高亮（角色名、唱段、技术提示、注释等）
- 实时预览渲染
- 译配模式切换
- Frontmatter 自动补全

安装后，`.dramark` 和 `.dra.md` 文件会自动识别为 DraMark 格式。

## Web 编辑器

Web 编辑器正在开发中，将提供在线编写和即时预览功能。

## 通用编辑器建议

DraMark 基于纯文本，任何支持 Markdown 的编辑器都可以编写。文件扩展名推荐使用 `.dramark` 或 `.dra.md`，编码使用 UTF-8。

## 输入法优化

DraMark 的核心符号（`@`、`$`、`%`、`=`）在中文键盘下可直接通过 Shift 键打出，无需切换输入法。这是 DraMark 的设计原则之一——保证编剧创作心流不断。
