# 注释

注释是 Attached Node，仅在源文件或特定工作版渲染目标中可见。

## 行注释

**语法**：`% 注释内容`

### 规则

- 行首第一个非空字符是 `%` → 整行为注释
- 行内注释：其左侧必须至少包含一个空白字符（即 `␠%`）

```dramark
% 这是注释

@角色
台词内容 % 这是普通文本，% 前无空格

台词内容 % 这是注释，% 前有空格
```

## 块注释

**语法**：`%%` 独占一行开始，`%%` 独占一行结束

```dramark
%%
此处为块注释
可包含多行内容

甚至可以包含空行
%%
```

## 使用场景

### 剧本注释

```dramark
% TODO: 这段台词需要修改

@角色
临时台词...
```

### 技术说明

```dramark
<<<
LX: SPOT_MAIN % 这是主要面光
LX: SPOT_FILL % 补光用
>>>
```

### 创作备忘

```dramark
%%
第一幕场景说明：
- 时间：深夜
- 地点：公园
- 情绪：忧郁
%%
```

## 注意事项

1. **块注释内**：所有 DraMark 语法均失效，视为纯文本
2. **代码保护区**：注释标记在代码块内也失效
3. **行尾附着**：角色声明行允许附着注释

```dramark
@小帕 [兴奋] % 检查麦克风状态
你好！
```
