# 转义字符

支持使用反斜杠 `\` 转义 DraMark 功能字符，使其降级为普通文本。

## 合法转义序列

| 序列 | 输出 |
|------|------|
| `\@` | `@` |
| `\$` | `$` |
| `\{` | `{` |
| `\}` | `}` |
| `\%` | `%` |
| `\<` | `<` |
| `\=` | `=` |
| `\>` | `>` |
| `` \` `` | `` ` `` |

## 使用示例

### 保留 @ 符号

```markdown
这是一个反斜杠 \@ 字符。 % 显示为 @，不触发角色声明
```

### 代码中的符号

```markdown
C++ 模板：vector\<int\> % 显示为 <int>，不触发 Tech Cue
位移操作：1 \<\< 8 % 显示为 <<，不触发 Tech Cue
```

### 保留百分号

```markdown
利润下降了 20\% % 显示为 20%，不是注释
```

### 保留美元符号

```markdown
价格是 \$50 % 显示为 $50，不是行内唱段
```

## 代码保护区内的转义

在代码块内，所有 DraMark 标记本身就失效，不需要转义：

````markdown
```cpp
// 这里不需要转义
vector<vector<int>> matrix;  // 正常显示
std::cout << "Hello" << std::endl;  // 正常显示
```
````

## 注意事项

1. **仅在 DraMark 语法有效的地方需要转义**
2. **代码保护区内无需转义**
3. **不支持的转义序列**：会保留反斜杠，如 `\a` 输出为 `\a`
